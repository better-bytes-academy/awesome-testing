## Cross-platform Testing

| Công cụ | Giá | Mô tả | Link |
|---------|-----|-------|------|
| **Appium** | Free | Open-source, automation cho iOS & Android | [appium.io](https://appium.io/) |
| **Detox** | Free | E2E testing cho React Native | [github.com/wix/Detox](https://github.com/wix/Detox) |
| **Maestro** | Free | UI testing đơn giản nhất cho mobile | [maestro.mobile.dev](https://maestro.mobile.dev/) |
| **Katalon Studio** | Free/Paid | All-in-one automation, hỗ trợ mobile | [katalon.com](https://www.katalon.com/) |
| **flutter_test** | Free | Testing framework cho Flutter apps | [flutter.dev](https://docs.flutter.dev/testing) |
