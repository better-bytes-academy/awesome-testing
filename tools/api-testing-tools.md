## API Testing Clients

| Công cụ | Giá | Mô tả | Link |
|---------|-----|-------|------|
| **Postman** | Free/Paid | Công cụ API testing phổ biến nhất, GUI trực quan | [postman.com](https://www.postman.com/) |
| **Insomnia** | Free/Paid | REST/GraphQL client nhẹ, dễ dùng | [insomnia.rest](https://insomnia.rest/) |
| **Bruno** | Free | Open-source, lưu trữ offline, git-friendly | [usebruno.com](https://www.usebruno.com/) |
| **Hoppscotch** | Free | Open-source API client trên web | [hoppscotch.io](https://hoppscotch.io/) |
| **Thunder Client** | Free | Extension cho VS Code | [thunderclient.com](https://www.thunderclient.com/) |
| **HTTPie** | Free/Paid | CLI & GUI API testing tool | [httpie.io](https://httpie.io/) |
| **curl** | Free | Command-line tool gửi HTTP request | [curl.se](https://curl.se/) |

## API Automation Frameworks

| Công cụ | Ngôn ngữ | Mô tả | Link |
|---------|---------|-------|------|
| **Rest Assured** | Java | Library phổ biến cho Java API testing | [rest-assured.io](https://rest-assured.io/) |
| **Karate** | Java | BDD-style API testing, không cần code Java | [github.com/karatelabs/karate](https://github.com/karatelabs/karate) |
| **Playwright** | Multi | API testing tích hợp trong E2E framework | [playwright.dev](https://playwright.dev/) |
| **SuperTest** | JavaScript | HTTP assertions cho Node.js | [github.com/ladjs/supertest](https://github.com/ladjs/supertest) |
| **requests** | Python | HTTP library đơn giản cho Python | [docs.python-requests.org](https://docs.python-requests.org/) |
| **RestQA** | JavaScript | BDD-based REST API testing | [github.com/restqa/restqa](https://github.com/restqa/restqa) |
| **Pactum** | JavaScript | REST API testing với BDD style | [pactumjs.github.io](https://pactumjs.github.io/) |

## API Mocking & Service Virtualization

| Công cụ | Mô tả | Link |
|---------|-------|------|
| **WireMock** | HTTP mock server mạnh mẽ cho Java | [wiremock.org](https://wiremock.org/) |
| **MockServer** | Mock HTTP server và proxy | [mock-server.com](https://www.mock-server.com/) |
| **JSON Server** | Tạo REST API fake từ JSON file | [github.com/typicode/json-server](https://github.com/typicode/json-server) |
| **Beeceptor** | No-code mock server, hỗ trợ REST/SOAP/GraphQL | [beeceptor.com](https://beeceptor.com/) |
| **Prism** | Mock server từ OpenAPI spec | [stoplight.io/prism](https://stoplight.io/prism) |
| **MSW** | Mock Service Worker cho browser/Node.js | [mswjs.io](https://mswjs.io/) |
