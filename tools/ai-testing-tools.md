## AI cho QA Workflow

### Cách sử dụng AI hiệu quả trong QA

| Use Case | Mô tả | Công cụ gợi ý |
|----------|-------|---------------|
| **Test case generation** | AI tạo test case từ requirements | ChatGPT, Copilot |
| **Bug analysis** | AI phân tích root cause | GitHub Copilot, Cursor |
| **Test data generation** | AI tạo test data đa dạng | Faker + AI, Synth |
| **Log analysis** | AI phân tích log tìm patterns | Datadog AI, Elastic |
| **Test maintenance** | AI tự sửa broken tests | Testim, mabl |
| **Code review** | AI review test code quality | Copilot, Cody |

### Prompt Engineering cho QA

```
Ví dụ prompt tạo test case:
"Generate test cases for a login form with the following fields:
- Email (required, valid format)
- Password (required, min 8 chars)
Include positive, negative, boundary, and edge cases."
```
