# ISTQB Guide

> Hướng dẫn ôn thi chứng chỉ ISTQB Foundation Level (CTFL).

---

## 📌 Giới thiệu ISTQB

**ISTQB** (International Software Testing Qualifications Board) là tổ chức quốc tế cấp chứng chỉ kiểm thử phần mềm được công nhận rộng rãi nhất trên thế giới.

### Các cấp độ chứng chỉ

```
Certified Tester — Expert Level (CTEL)
    ↑
Certified Tester — Advanced Level (CTAL)
├── Test Manager
├── Test Analyst
└── Technical Test Analyst
    ↑
Certified Tester — Foundation Level (CTFL) ← Bắt đầu từ đây
```

---

## 📖 Nội dung CTFL v4.0 (2023)

### Chương 1: Fundamentals of Testing (180 phút)
- Kiểm thử phần mềm là gì và tại sao cần thiết?
- 7 nguyên tắc kiểm thử
- Quy trình kiểm thử cơ bản
- Tâm lý kiểm thử

### Chương 2: Testing Throughout SDLC (130 phút)
- Testing trong SDLC models
- Test levels và test types
- Maintenance testing

### Chương 3: Static Testing (80 phút)
- Static testing basics
- Review process
- Review types (Informal, Walkthrough, Technical, Inspection)

### Chương 4: Test Analysis and Design (390 phút)
- Test design techniques
- Black-box: EP, BVA, Decision Table, State Transition
- White-box: Statement, Branch Coverage
- Experience-based: Error Guessing, Exploratory, Checklist-based

### Chương 5: Managing the Test Activities (335 phút)
- Test planning
- Risk management
- Test monitoring & control
- Configuration management
- Defect management

### Chương 6: Test Tools (20 phút)
- Tool support for testing
- Benefits & risks of test automation

## 💡 Tips ôn thi

1. **Đọc kỹ Syllabus chính thức** — nguồn ôn thi quan trọng nhất
2. **Làm mock exams** — quen với dạng câu hỏi
3. **Hiểu K-levels** — mỗi chương yêu cầu mức hiểu khác nhau:
   - K1: Remember
   - K2: Understand
   - K3: Apply
4. **Focus Chương 4** — chiếm nhiều câu hỏi nhất
5. **Đọc câu hỏi thật kỹ** — ISTQB hay đánh lừa bằng từ ngữ
