# QA Engineer Roadmap

> Lộ trình học tập và phát triển cho QA Engineer — từ Fresher đến Senior.

---

## 🗺️ Roadmap tổng quan

```
Fresher (0-1 năm)
├── Testing Fundamentals
├── SDLC & STLC
├── Test Case Design
├── Bug Reporting
├── Basic SQL
└── Agile/Scrum

Junior (1-2 năm)
├── Test Design Techniques (EP, BVA, Decision Table)
├── API Testing (Postman, REST)
├── Basic Automation (Selenium/Playwright basics)
├── CI/CD Basics
├── Version Control (Git)
└── ISTQB Foundation

Mid-level (2-4 năm)
├── Automation Framework Design (POM)
├── Programming (Python/Java/JavaScript)
├── Performance Testing Basics
├── Security Testing Basics
├── Database Testing
├── Mobile Testing
└── Docker Basics

Senior (4+ năm)
├── Test Strategy & Architecture
├── Advanced Automation
├── CI/CD Pipeline Design
├── Performance Engineering
├── Security Testing
├── Team Leadership
├── Mentoring
└── Process Improvement
```

---

## 📚 Fresher (0-1 năm)

### Kiến thức cần học

| Chủ đề | Tài nguyên |
|--------|-----------|
| Testing Fundamentals | ISTQB Foundation Syllabus |
| SDLC / STLC | Software Testing Help |
| Test Case Writing | Guru99 |
| Bug Reporting | Jira / Azure DevOps |
| Basic SQL | W3Schools SQL |
| Agile / Scrum | Scrum Guide |
| Linux Basics | Command line basics |

### Kỹ năng mềm
- Communication — báo cáo rõ ràng
- Attention to detail — tỉ mỉ, cẩn thận
- Critical thinking — tư duy phản biện
- Documentation — viết tài liệu

---

## 🚀 Junior (1-2 năm)

### Kiến thức cần học

| Chủ đề | Tài nguyên |
|--------|-----------|
| Test Design Techniques | EP, BVA, Decision Table, State Transition |
| API Testing | Postman, REST API concepts |
| Basic Automation | Selenium/Playwright cơ bản |
| Git | Git basics, branching, PR |
| CI/CD Basics | GitHub Actions, Jenkins basics |
| ISTQB Foundation | Thi chứng chỉ |

### Tools nên biết
- **Test management**: Jira, TestRail
- **API testing**: Postman, Bruno
- **Browser DevTools**: Network, Console, Elements
- **Version control**: Git, GitHub/GitLab

---

## 💪 Mid-level (2-4 năm)

### Kiến thức cần học

| Chủ đề | Tài nguyên |
|--------|-----------|
| Programming | Python / Java / JavaScript |
| Automation Framework | POM, Data-driven |
| Playwright / Cypress | Advanced features |
| Performance Testing | JMeter / k6 basics |
| Security Testing | OWASP Top 10 |
| Database Testing | SQL advanced queries |
| Mobile Testing | Appium basics |
| Docker | Containers, Docker Compose |

### Kỹ năng mềm
- Mentoring junior members
- Estimation techniques
- Risk assessment
- Stakeholder communication

---

## 🏆 Senior (4+ năm)

### Kiến thức cần học

| Chủ đề | Tài nguyên |
|--------|-----------|
| Test Strategy | Risk-based testing |
| Architecture | Microservices testing |
| CI/CD Pipeline | Full pipeline design |
| Performance Engineering | Load, Stress, Soak testing |
| Security | Penetration testing basics |
| Cloud | AWS/Azure/GCP basics |
| Leadership | Team management, process improvement |

### Certifications gợi ý
- ISTQB Advanced Level
- AWS Certified Cloud Practitioner
- Certified Scrum Master (CSM)
