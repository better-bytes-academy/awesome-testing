## Các vị trí trong QA

| Vị trí | Kinh nghiệm | Mô tả |
|--------|-------------|-------|
| **QA Intern / Fresher** | 0-1 năm | Học testing cơ bản, thực thi test cases |
| **Junior QA** | 1-2 năm | Manual testing, viết test cases, report bugs |
| **QA Engineer** | 2-4 năm | Manual + Automation, API testing |
| **Senior QA / SDET** | 4-6 năm | Framework design, test strategy, CI/CD |
| **QA Lead** | 5-8 năm | Lead team, planning, process improvement |
| **QA Manager** | 7+ năm | Quản lý nhiều team/projects, strategy |
| **QA Architect** | 8+ năm | Test architecture, tool selection, standards |
| **Head of QA / VP QA** | 10+ năm | Strategic direction, organization |

## Cách viết CV QA

### Cấu trúc CV

1. **Contact Info** — Tên, email, phone, LinkedIn, GitHub
2. **Summary** — 2-3 câu tóm tắt kinh nghiệm
3. **Skills** — Technical & soft skills
4. **Experience** — Mô tả công việc + thành tích cụ thể
5. **Education** — Bằng cấp
6. **Certifications** — ISTQB, AWS, etc.
7. **Projects** — Side projects, open source

### Tips viết CV

✅ **Nên:**
- Dùng số liệu cụ thể: "Automated 200+ test cases, reducing regression time by 60%"
- Liệt kê tools & technologies đã dùng
- Highlight achievements, không chỉ responsibilities
- Customize CV cho từng vị trí

❌ **Không nên:**
- Liệt kê quá dài, không relevant
- Dùng generic descriptions
- Sai chính tả / grammar

### Action verbs cho QA CV
- Designed, Implemented, Automated, Identified, Reduced, Improved, Led, Mentored, Analyzed, Developed, Executed, Reported

## Con đường phát triển

### Track 1: Technical (IC Track)
```
Junior QA → QA Engineer → Senior QA → SDET → Principal QA → QA Architect
```

### Track 2: Management Track
```
Junior QA → QA Engineer → Senior QA → QA Lead → QA Manager → Head of QA
```

### Track 3: Chuyển đổi
```
QA → DevOps Engineer
QA → Product Manager
QA → Business Analyst
QA → Scrum Master
QA → Full-stack Developer
```
